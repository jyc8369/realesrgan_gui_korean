# Real-ESRGAN GUI (Korean Edition)

이 프로젝트는 [TransparentLC/realesrgan-gui](https://github.com/TransparentLC/realesrgan-gui)의 **포크 버전**입니다.

## 주요 변경사항

### 한국어 지원 추가
- 사용자 인터페이스 (UI)를 한국어로 현지화
- 메뉴, 버튼, 메시지 등 모든 텍스트 한국어 지원
- i18n (국제화) 설정 파일 포함

## 설명

Real-ESRGAN GUI는 사진 및 비디오를 업스케일링하는 크로스 플랫폼 GUI 애플리케이션입니다.
